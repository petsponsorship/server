## 💻 구현 기능

- 게시글 CRUD
- 로그인, 회원가입
- JWT 활용한 토큰 관리(Refresh Token, Access Token)
- 게시글 후원하기
- 게시글 좋아요
- 마이페이지 (내가 쓴 글, 후원한 글, 좋아요한 글)
- `multer` 패키지 활용 - 이미지 S3 에 저장
- `node-schedule` 패키지 활용 - 게시 후 2주가 경과한 글은 자동 만료
